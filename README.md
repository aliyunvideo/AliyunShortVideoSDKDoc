
# 概述

阿里云短视频SDK（[进入官网](https://promotion.aliyun.com/ntms/act/shortvideo.html)）分为基础、标准、专业三个版本。短视频SDK提供短视频录制、导入和编辑的功能。为您提供产品级的UI开源界面，可以根据自己的业务定制界面，提供易用、稳定、统一的视频录制、导入裁剪和编辑高级接口，实现真正的二次开发、做到真正个性化。
