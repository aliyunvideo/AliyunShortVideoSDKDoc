
# 视频编辑

短视频SDK提供视频编辑与导出功能，支持视频图片素材混合导入、提供滤镜、配音、时间特效、过渡效果等丰富的编辑效果。


## 版本支持


|    版本    | 基础版 | 标准版 | 专业版 |
| :--------: | :----: | :----: | :----: |
|  导入视频  |   x    |   ✓    |   ✓    |
|  导出视频  |   x    |   ✓    |   ✓    |
|  草稿箱  |   x    |   ✓    |   ✓    |
|  视频管理  |   x    |   ✓    |   ✓    |
| 音频音效 |   x    |   ✓    |   ✓    |
| 滤镜及转场 |   x    |   ✓    |   ✓    |
|  时间特效  |   x    |   ✓    |   ✓    |
|   画中画   |   x    |     ✓   |   ✓    |
|    字幕    |   x    |   x    |   ✓    |
|  动态贴纸  |   x    |   x    |   ✓    |
|     MV     |   x    |   x    |   ✓    |
