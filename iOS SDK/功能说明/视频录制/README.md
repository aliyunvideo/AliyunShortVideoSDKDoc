# 视频录制

视频录制有三种类型： 普通录制、视频合拍、多元录制。

- [基础录制](基础录制.md):  摄像头采集录制功能
- [视频合拍](视频合拍.md):  使用一个已有视频作为样本视频，与摄像头采集的数据按照特定的布局方式,进行合拍录制
- [多源录制](多源录制.md):  可以支持View录制、摄像头录制等多种视频采集源按需组合的合拍录制
